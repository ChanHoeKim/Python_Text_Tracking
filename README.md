## 1. 프로젝트 개요

- **프로젝트명**: EasyOCR 기반 화면 텍스트 인식 자동 카운팅 프로그램
- **기간**: 개인 프로젝트
- **개발환경/기술스택**:
    - Python
    - PyQt6
    - EasyOCR
    - OpenCV(cv2)
    - MSS(Screen Capture)
    - NumPy
    - JSON 설정 저장
- **프로젝트 목적**
    
    → 화면 내 특정 영역의 텍스트를 실시간 OCR로 인식하고, 유사 문자열 매칭을 통해 자동 카운팅 및 수익 계산 기능 제공
    
    → 게임 및 반복 작업 환경에서 수작업 기록을 자동화하기 위한 OCR 기반 유틸리티 개발
    

---

## 2. 주요 기능

### (1) OCR 텍스트 인식

- **실시간 화면 OCR**
    - MSS를 활용해 지정 영역(Screen Capture) 실시간 캡처
    - EasyOCR 기반 한글·영문 텍스트 인식
- **전처리 기능**
    - OpenCV 기반 이미지 그레이스케일 변환
    - 해상도 확대 및 대비 조절로 OCR 정확도 향상
- **텍스트 색상 분석**
    - HSV 기반 색상 추출
    - 특정 색상 제외 기능 구현
    - 색상 유사 범위(Tolerance) 적용으로 안정성 강화

---

### (2) 자동 카운팅 시스템

- **유사 문자열 매칭**
    - `SequenceMatcher` 활용
    - OCR 오인식 발생 시에도 유사 텍스트 자동 판별
- **중복 카운팅 방지**
    - 최근 인식 캐시 저장
    - 텍스트·색상·Y좌표·시간 조건을 조합하여 중복 탐지 차단
- **유사도 스코어 설정**
    - 사용자가 직접 OCR 매칭 민감도 조절 가능
    - UI에서 실시간 변경 지원

---

### (3) UI 및 사용자 편의 기능

- **투명 인식 영역 UI**
    - PyQt6 기반 드래그·리사이즈 가능한 오버레이 구현
    - 항상 위(WindowStaysOnTopHint) 옵션 적용
- **실시간 결과 표시**
    - OCR 결과 및 매칭 점수 실시간 출력
    - 상태 메시지 제공(인식 여부, 오류 상태 등)
- **테이블 기반 계산 기능**
    - 품목별 수량·금액·최종 금액 자동 계산
    - 전체 합계 자동 갱신
- **시간당 평균 계산**
    - 1시간 단위 수익 기록
    - 평균 수익 자동 계산 기능 구현

---

## 3. 차별화된 기술 포인트

- **OCR 정확도 개선**
    
    → OpenCV 전처리(확대·명암 조절)를 적용하여 인식률 향상
    
- **유사 문자열 기반 자동 매칭**
    
    → 단순 문자열 비교가 아닌 Similarity 기반 비교로 OCR 오차 대응
    
- **중복 탐지 최적화**
    
    → 텍스트·좌표·색상·시간을 조합한 캐시 로직으로 중복 카운팅 최소화
    
- **PyQt6 기반 커스텀 UI**
    
    → 투명 오버레이 및 드래그·리사이즈 기능 직접 구현
    
- **설정 저장 기능**
    
    → JSON 파일 기반으로 UI 위치·테이블 정보·OCR 설정 자동 저장 및 복원

## 4. 기대 효과 및 학습 성과

- **OCR 기반 자동화 프로그램 개발 경험**
    
    → 화면 캡처 → OCR → 데이터 처리 → UI 출력까지 전체 흐름 구현 경험 확보
    
- **실시간 데이터 처리 경험**
    
    → Thread(QThread) 기반 비동기 OCR 처리 및 UI 업데이트 구조 설계
    
- **컴퓨터 비전 활용 경험**
    
    → OpenCV 이미지 전처리 및 색상 분석 로직 구현
    
- **사용자 중심 기능 설계**
    
    → 실시간 설정 변경, 중복 방지, 자동 저장 등 실제 사용성을 고려한 기능 구현
    
- **데스크톱 애플리케이션 개발 경험**
    
    → PyQt6 기반 GUI 설계 및 이벤트 처리 구조 학습
